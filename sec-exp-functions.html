    Section 3.1 Exponential Functions

    Definition 3.1.1.

    An exponential function is a function that can be written in the form
    \begin{equation*} f(x) = a^x \end{equation*}
    where \(a\) is a positive number (not equal to 1) called the base of the function.
    🔗
    🔗
    🔗
    The behavior of the function depends on the value of the base, \(a\text{.}\) For example, if \(a=2\text{,}\) the graph \(y = 2^x\) looks like
    🔗
    We can see several properties of the function \(y=2^x\) from its graph. It’s domain is \((-\infty, \infty)\text{.}\) This is because, no matter what we choose for the value of \(x\text{,}\) the expression \(2^x\) is always defined. The graph’s range does not contain any negative numbers. In fact, \(y=0\) is an asymptote of the graph, and the range is \((0, \infty)\text{.}\) The graph does not have any \(x\)-intercepts, reflecting the fact that the equation \(2^x = 0\) does not have any solutions. The graph has a \(y\)-intercept at \(y=1\text{,}\) because \(f(0) = 2^0 = 1\text{.}\)
    🔗
    When we consider the end behavior of the graph, we can see that as \(x\) moves to the left on the graph, the \(y\)-values get closer to the horizontal asymptote. We can write this concisely as
    \begin{gather*} y \rightarrow 0 ~\text{as}~ x \rightarrow -\infty. \end{gather*}
    On the other side, we can see that as \(x\) moves to the right on the graph, the \(y\)-values get larger and larger. So we can write
    \begin{gather*} y \rightarrow \infty ~\text{as}~ x \rightarrow \infty. \end{gather*}
    In fact, as \(x\) moves to the right, the \(y\)-values are growing extremely quickly. Consider this table of values. Notice that for each step when the \(x\)-value increases by one, the \(y\)-value doubles. This doubling quickly leads to very large \(y\)-values. For example, if we continued our table a few more steps to \(x=10\text{,}\) the corresponding \(y\)-value would be \(f(10) = 2^10 = 1024\text{.}\) This type of rapid growth occurs whenever the base of the exponential function is greater than 1, and is called exponential growth.
