    Section 1.1 Definitions

    As we move through Precalculus and Calculus, we will need to be careful with the words that we use. Mathematical writing is characterized by precise and careful language, which allows us to avoid ambiguity and possible misunderstandings. However, we will only avoid misunderstanding if we all know what words mean. Throughout these notes, I will include definitions for the terms that I use, and I need you to make note of these definitions and refer back to them when you see a word you don’t recognize. In many ways, mathematics is a language that can describe quantitative or geometric information. Calculus, the focus of these notes, is a language that we can use to describe quantities that change. We will spend a lot of time discussing the rules that govern this language, but it is also crucially important to take the time to learn the words that make up this language.
    🔗

    Definition 1.1.1. Expression.

    An expression is a combination of numbers, variables, and operations.
    🔗
    🔗
    This definition perhaps leads to more questions than it answers. After all, what are the precise, mathematical definitions of numbers, variables, and operations?
    🔗
    For us, in this book, numbers will always refer to real numbers. It is difficult to define real numbers precisely, but you are familiar with them nonetheless. These are numbers that correspond to positions on the number line. Positive real numbers correspond to lengths. All real numbers are positive, negative, or zero. Real numbers include whole numbers, fractions, and irrational numbers. Irrational numbers, like \(\pi\) and \(\sqrt{2}\text{,}\) are numbers that can’t be written as ratios of whole numbers, and their decimal representations are nonterminating and nonrepeating.
    🔗
    A variable is a letter or symbol that is used to represent an unknown or unspecified number. There is some nuance here. In an equation, the variable is unknown and the goal will be to solve for all possible values of the variable that make the equation true. In function notation, the variable is unspecified, and we will replace the variable with various numbers and see how the output of the function changes.
    🔗
    An operation is a rule for transforming one or more numbers into a new number. Most operations act on two numbers, and examples of operations include addition, subtraction, multiplication, and division. Importantly, equality (\(=\)) is not an operation, but instead something called a relation. So, while \(15x^2+3x\) and \(\frac{9x-3}{2}\) are examples of expressions, the equation \(3x+2 = 6x -1\) is not an expression.
    🔗

    Definition 1.1.2. Equation.

    An equation is statement that the values of two mathematical expressions are equal.
    🔗
    🔗
    An equation might always be true, like the equations \(2+2=4\) and \(\ln \left( e^3 \right) = 3\text{,}\) or they might be conditionally true, such as the equation \(5x+3x^2 = 2\text{,}\) which is only true if \(x=-2\) or \(x = \frac{1}{3}\text{.}\) It is also possible for an equation to always be false, such as the equation \(\sin^2(x)+\cos^2(x) = 0\text{.}\)
    🔗

    Definition 1.1.3. Inequality.

    An inequality is a statement that compares two expressions using one of the following symbols: \(<\) (less than), \(\leq\) (less than or equal to), \(>\) (greater than), \(\geq\) (great than or equal to), or \(neq\) (not equal).
    🔗
    🔗
    Just like equations, an inequality might be always true, conditionally true, or always false. For example, the inequality \(x^2 \geq 0\) is true for every real number \(x\) because the square of a real number is always zero or positive. On the other hand, the inequality \(x^2 < x\) is only true if the number \(x\) is between 0 and 1.
    🔗

    Remark 1.1.4. Notes about inequalities.

    • The statement \(X > 0\) means that the expression \(X\) is positive. On the other hand, the statement \(X < 0\) means that the expression \(X\) is negative.
      🔗
      🔗
    • Geometrically, saying \(a < b\) means that the number \(a\) is to the left of the number \(b\) on the number line, while the statement \(a > b\) means that \(a\) is to the right of \(b\) on the number line.
      🔗
      🔗
    • Every inequality written using "\(<\)" (or "\(\leq\)") can be rewritten using "\(>\)" (or "\(\geq\)") and vice versa. For example, the inequality \(-5 < -2\) is equivalent to \(-2 > -5\text{.}\) By the way, can you tell if these inequalities are true or false? If you’re not sure, try using a number line.
      🔗
      🔗
    🔗
    🔗

    Definition 1.1.5. Set.

    A set is a collection of numbers (or other mathematical objects).
    🔗
    🔗
    Think of a set as a box that you can store numbers in. The numbers stored in a set are called the elements of that set. For example, the set containing the numbers 1, 3, and 5 would be written as \(\{ 1, 3, 5\}\text{.}\) This is a finite set containing three elements. On the other hand, the set of even integers would contain every multiple of 2. This set could be written as \(\{ \ldots, -4, -2, 0, 2, 4, 6, \ldots \}\text{,}\) where the dots indicate that the pattern would continue in both directions. This is an infinite set, because no matter how many even numbers we list inside the brackets, we can never list them all. We will not spend much time on sets in these courses. If you are interested in learning more about set theory, you should take our Introduction to Mathematical Thinking course.
